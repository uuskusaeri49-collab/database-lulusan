Pengaturan Backup & Pemulihan

Cadangkan Database (Backup)

Gunakan fitur ini untuk mengekspor seluruh catatan lulusan, riwayat, berkas digital, dan parameter sistem dalam bentuk file portabel berformat `.json`.

Pulihkan Database (Restore)

Gunakan fitur ini untuk memuat kembali berkas cadangan `.json` yang telah Anda simpan sebelumnya. Seluruh data saat ini akan digantikan oleh data cadangan tersebut.
